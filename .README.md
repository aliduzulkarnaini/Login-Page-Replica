# Login Page Replica

This project is a static login page built with HTML and CSS. The page recreates a modern login card design with a centered layout, a purple gradient background, layered geometric card shapes, username and password fields, a login button, and social login icons.

## About the Page

The page is made up of two main files:

- `index.html` contains the structure of the login form, input fields, button, background shape elements, and social icons.
- `style.css` contains the full visual design, including the gradient background, card layout, positioning, colors, icons, shadows, and responsive sizing.

The design uses a single centered card with decorative rotated shapes behind the form. The form includes simple username/email and password fields, a styled login button, and social media icon options for Instagram, Facebook, and Twitter.

## What I Learnt

While completing this assignment, I learnt how to turn a visual design into an actual web page using HTML and CSS. I practiced creating a clean HTML structure and connecting it to an external stylesheet.

I also learnt more about CSS positioning, especially how `position: absolute` and `position: relative` can be used together to place elements exactly where they are needed. This helped me build the layered card background and place the form and social login section on top of it.

Another important thing I learnt was how to use CSS properties like `border-radius`, `box-shadow`, `transform`, `z-index`, and `linear-gradient` to create a more polished design. I also practiced using flexbox to center the page content and align items inside the form and button.

Overall, this assignment helped me improve my understanding of page layout, visual styling, and how small CSS details can make a login page look more professional.
